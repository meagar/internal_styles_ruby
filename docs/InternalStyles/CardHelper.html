    Generate a Bootstrap card with the given header

    Signatures¶ ↑

  # card body
end

# with no header

    Options¶ ↑

    • :id - The ID of the outer <div class=“card”> element

    • :class - Additional classes to add to the <div class=“card”> element

    • :body_class - Additional classes to add to the <div class=“card-body”> element

    • :header_class - Additional classes to add to the <div class=“card-header”> element

  This is the body of the test card
<% end %>

  Test Body

def card(header = nil, **options, &block)
  if header.is_a?(Hash) && options.empty?
    options = header
    header = nil
  end

  classes = ['card mb-3'] + Array.wrap(options[:class])
  header_classes = ['card-header'] + Array.wrap(options[:header_class])
  body_classes = ['card-body'] + Array.wrap(options[:body_class])

  id = options[:id]

  body_proc = options[:body_proc] || Proc.new do |body_classes|
    content_tag(:div, class: body_classes, &block)
  end

  card_tag(id: id, class: classes) do
    buffer = []
    buffer << content_tag(:div, header, class: header_classes) if header
    buffer << body_proc.call(body_classes)

    safe_join(buffer)
  end
end

def card_for(record, header = nil, **options, &block)
  raise unless block_given?
  if header.is_a?(Hash) && options.empty?
    options = header
    header = nil
  end

  options[:id] = options.fetch(:id, dom_id(record))
  options[:class] = [dom_class(record)] + Array.wrap(options[:class])

  card(header, **options, &block)
end
    card_tag(**options, &block) click to toggle source

    Generate a Bootstrap Card tag in the form <div class="card">

    Signatures¶ ↑

    card_tag(**options) do
  # content for card
end

    Options¶ ↑

    The options argument is passed directly through to content_tag, see api.rubyonrails.org/classes/ActionView/Helpers/TagHelper.html#method-i-content_tag.

    The sole exception is that the :class option (if any) has the bootstrap class card added to it.

    Examples¶ ↑

    Card with default options:

    <%= card_tag do %>
  <p>Card content</p>
<% end %>
# => <div class="card">
       <p>Card content</p>
     </div>

    Card with additional options (note card is still added to the class list):

    <%= card_tag(id: 'my-card', class: 'large-card') do %>
  <p>Card content</p>
<% end %>
# => <div id="my-card" class="card large-card">
       <p>Card content</p>
     </div>
